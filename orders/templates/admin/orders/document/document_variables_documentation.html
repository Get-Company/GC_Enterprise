Template Variablen Dokumentation

Diese Dokumentation beschreibt die verfügbaren Variablen für das Document, Projekt und Kunden, sowie deren Attribute. Du kannst diese Variablen in deinen Templates mit {{ }} verwenden.

Allgemeine Syntax

  • {{ variable }}: Gibt den Wert einer Variable aus.
  • {% if condition %}: Überprüft eine Bedingung. Nutze {% endif %}, um den Block zu schließen.

Modelle und ihre Attribute

1. Document

Repräsentiert ein Dokument innerhalb eines Projekts.

{{ document.order }}: Das zugehörige Projekt (vom Typ Order).

{{ document.document_type }}: Der Dokumenttyp (z. B. "Rechnung").

{{ document.file_path }}: Der Pfad zur Datei.

{{ document.introduction }}: Einleitungstext des Dokuments.

{{ document.description }}: Beschreibung des Dokuments.

{{ document.conclusion }}: Abschluss des Dokuments.

2. Projekt

Repräsentiert ein Projekt innerhalb des Systems.

{{ order.name }}: Der Name des Projekts.

{{ order.nr }}: Die Auftragsnummer.

{{ order.customer }}: Der zugehörige Kunde (vom Typ Customer).

{{ order.description }}: Beschreibung des Projekts.

{{ order.hourly_rate }}: Der Stundensatz.

{{ order.flat_rate }}: Die Pauschale.

{{ order.total_net }}: Gesamtnetto-Betrag.

{{ order.total_vat }}: Mehrwertsteuerbetrag.

{{ order.total_gross }}: Gesamtbruttobetrag.

3. Customer

Repräsentiert einen Kunden.

{{ customer.name }}: Der Name des Kunden.

{{ customer.street }}: Die Straße.

{{ customer.house_number }}: Die Hausnummer.

{{ customer.postal_code }}: Die Postleitzahl.

{{ customer.city }}: Die Stadt.

{{ customer.email }}: Die E-Mail-Adresse.

{{ customer.phone }}: Die Telefonnummer.

{{ customer.vat_id }}: Die Umsatzsteuer-Identifikationsnummer.

Beispiele

Du kannst die Variablen in deinen Templates wie folgt verwenden:

{% if order.total_net > 1000 %}Das Projekt ist groß.{% endif %}
{{ customer.name }} aus {{ customer.city }} hat das Dokument {{ document.document_type }} erhalten.
